Tổng quan định tuyến (Routing)

Toàn bộ định tuyến trong dự án được tổ chức tách biệt theo mô hình hướng đối tượng: các Router trung gian cấu hình đường dẫn và kiểm tra quyền truy cập (Middlewares), sau đó điều phối logic xử lý đến lớp Controller tương ứng.

Authentication API (Xác thực người dùng)

GET /register Khách (Guest)
Hiển thị giao diện HTML trang đăng ký tài khoản cho sinh viên mới.
POST /register Khách (Guest)
Xử lý biểu mẫu đăng ký. Băm (hash) mật khẩu và lưu tài khoản mới vào MongoDB.
Trường Kiểu Bắt buộc Mô tả
username String Có Tên hiển thị của sinh viên.
email String Có Email đăng nhập (không trùng lặp).
password String Có Mật khẩu của tài khoản.
GET /login Khách (Guest)
Hiển thị giao diện HTML trang đăng nhập hệ thống.
POST /login Khách (Guest)
Kiểm tra thông tin đăng nhập, thiết lập Session cho phiên làm việc của người dùng.
Trường Kiểu Bắt buộc Mô tả
email String Có Email tài khoản.
password String Có Mật khẩu tài khoản.
GET /logout Yêu cầu Đăng nhập
Hủy bỏ Session của phiên làm việc hiện tại và điều hướng người dùng về trang đăng nhập.
📂

Folders API (Quản lý Thư mục)

GET /folders Yêu cầu Đăng nhập
Hiển thị toàn bộ danh sách thư mục gốc thuộc sở hữu của người dùng đang đăng nhập.
GET /folder/:id Yêu cầu Đăng nhập
Hiển thị danh sách các thư mục con và các tệp tài liệu nằm bên trong một thư mục cụ thể.
Tham số URL Kiểu Bắt buộc Mô tả
id ObjectId Có Mã ID định danh duy nhất của thư mục cần truy xuất.
POST /folder/create Yêu cầu Đăng nhập
Tạo thư mục học tập mới. Hỗ trợ tạo thư mục lồng nhau.
Trường Body Kiểu Bắt buộc Mô tả
name String Có Tên thư mục muốn tạo mới.
parentFolder ObjectId / Null Không ID của thư mục cha. Nếu để trống hoặc null, thư mục sẽ được tạo ở ngoài cùng (Root).
POST /folder/:id/rename Yêu cầu Đăng nhập
Đổi tên thư mục học tập hiện có.
Tham số Kiểu Bắt buộc Mô tả
id (URL) ObjectId Có ID của thư mục cần đổi tên.
name (Body) String Có Tên mới muốn cập nhật cho thư mục.
POST /folder/:id/delete Yêu cầu Đăng nhập
Xóa thư mục và thực hiện **Xóa bắc cầu (Cascade Delete)** toàn bộ các thư mục con và các file nằm bên trong nó (xóa tệp vật lý trên S3 và xóa bản ghi trên MongoDB).

Documents API (Quản lý Tài liệu)

GET /upload Yêu cầu Đăng nhập
Hiển thị giao diện trang tải lên tài liệu học tập mới.
POST /upload Yêu cầu Đăng nhập
Tải tệp tin lên AWS S3 (thông qua Multer đệm bộ nhớ) và ghi nhận siêu dữ liệu tệp vào MongoDB Atlas.
Trường Kiểu truyền tải Bắt buộc Mô tả
file multipart/form-data Có Tệp tin tài liệu bài giảng gửi đính kèm từ máy tính.
folderId String / Null Không ID thư mục đích muốn lưu tệp vào. Nếu trống, lưu ở thư mục gốc.
GET /document/:id Yêu cầu Đăng nhập
Hiển thị trang chi tiết tài liệu, hỗ trợ khung đọc tài liệu PDF/Hình ảnh trực tiếp và khung Trợ lý ảo AI Chatbot học tập.
GET /document/:id/download Yêu cầu Đăng nhập
Máy chủ Express nhận diện Key từ S3, tạo luồng dữ liệu truyền dẫn an toàn (Piping Stream) về trình duyệt của người dùng để xem trực tuyến hoặc lưu về đĩa.
DELETE /document/:id Yêu cầu Đăng nhập
Xóa tệp tin vật lý tương ứng trên AWS S3 và đồng thời xóa thông tin siêu dữ liệu của tệp trong MongoDB.
GET /search Yêu cầu Đăng nhập
Tìm kiếm toàn văn các thư mục hoặc tài liệu thuộc sở hữu của người dùng hiện tại thông qua từ khóa tìm kiếm.
Query Parameter Kiểu Bắt buộc Mô tả
q String Có Từ khóa tìm kiếm (tìm khớp tên thư mục hoặc tên tệp tài liệu).
🤖

AI Assistant Chat API (Trợ lý ảo AI)

POST /ai/chat Yêu cầu Đăng nhập
Gửi tin nhắn trao đổi hoặc câu hỏi thắc mắc về một tài liệu bài giảng học tập cụ thể. Server sẽ nạp nội dung tài liệu từ AWS S3 và chuyển tiếp làm ngữ cảnh nền (Context Window) cho Google Gemini API để trả về câu trả lời chuẩn xác.
Trường Body Kiểu Bắt buộc Mô tả
message String Có Nội dung câu hỏi của sinh viên.
documentId ObjectId Có Mã ID tài liệu học tập đang mở đọc để AI có thể đọc hiểu ngữ cảnh tài liệu đó.
